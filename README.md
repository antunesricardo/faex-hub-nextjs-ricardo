🚀 Como Executar

Pré-requisitos

•
Node.js 18+ instalado

•
npm ou yarn instalado

Instalação

Bash


npm install
# ou
yarn install



Desenvolvimento

Bash


npm run dev
# ou
yarn dev



Abra http://localhost:3000 no seu navegador para ver o resultado.

Build para Produção

Bash


npm run build
npm start
# ou
yarn build
yarn start
